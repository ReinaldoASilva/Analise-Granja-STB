
# Controle de Temperatura e Umidade no Aviário

" Este projeto de análise de dados tem como objetivo investigar como as condições ambientais afetam o desempenho das aves.Analisaremos os momentos em que a temperatura e umidade atingem niveis extremos, buscando compreender como esses fatores influenciam na conversão alimentar.
A ideia é buscar melhorias nas práticas de manejo e identificar os intervalos perfeitos de temperatura e umidade, com o objetivo de maximizar os lucros nas granjas analisadas."

# Tecnologias Utilizadas
Python 3.7 ou superior
Streamlit - versão 1.15.0
Pandas - versão 1.5.1
Plotly - versão
PIL (Python Imaging Library)

## Instalação e Execução

1. Certifique-se de ter o Python 3.7 ou uma versão mais recente instalada em seu sistema.

2. Clone o repositório para o seu ambiente local.

3. Abra o terminal e navegue até o diretório do projeto.

4. Crie um ambiente virtual (opcional, mas recomendado) e ative-o.

5. Instale as dependências listadas no arquivo `requirements.txt` usando o comando:

   ````
   pip install -r requirements.txt
   ```

6. Execute o aplicativo Streamlit com o comando:

   ````
   streamlit run main.py
   ```

7. O aplicativo será iniciado e você poderá acessá-lo no navegador pelo endereço `localhost:8501`.

## Funcionalidades

- Visualização dos dados de temperatura e umidade no aviário.
- Sugestões e medidas para melhorar o controle da temperatura e umidade.
- Monitoramento em tempo real das condições do aviário.
- Análise histórica dos dados para identificar padrões e tendências.

## Estrutura do Projeto

- `Home.py`: Arquivo principal que contém a lógica do aplicativo Streamlit.
- `Analise-Granja-STB/`: Pasta contendo os arquivos de dados históricos do aviário.
- `pages/`: Pasta contendo as imagens utilizadas no aplicativo.
- `requirements.txt`: Arquivo com as dependências necessárias para executar o projeto.
